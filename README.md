# Sprout — Plant Care Dashboard

A plant care app dashboard built as a design exercise. Sprout helps users track their plants, monitor care schedules, and connect with a community of fellow plant owners.

## Features

- **Your Plants** — 6 plant cards with status indicators (healthy, water soon, needs attention, just added)
- **Care Alerts** — urgent care notifications for plants that need attention
- **Community** — recent activity from other Sprout users
- **Responsive sidebar** — primary and secondary navigation with Tabler icons

## Brand

- **Colors:** Midnight navy `#0F1B2D`, Terracotta `#B85C38`, Sage `#7A9E87`, Off-white `#F7F4EF`, Stone `#E8E0D4`
- **Fonts:** DM Serif Display (logo), DM Sans (everything else)
- **Icons:** [Tabler Icons](https://tabler-icons.io)
